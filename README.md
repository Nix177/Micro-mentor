# Application 2 : Micro-Mentorat Instantané

> "L'information est abondante, mais l'aide spécifique et immédiate est rare."

## Vision
Une plateforme de **micro-mentorat** conçue pour résoudre des problèmes bloquants en **3 minutes ou moins**. 

## Piliers Clés
1.  **Immédiateté** : Pas de planification. "J'ai un problème maintenant, j'ai besoin d'aide maintenant."
2.  **Format Court** : Sessions limitées strictement à 3-5 minutes pour réduire la charge mentale des mentors.
3.  **Réciprocité** : Gagnez des crédits en aidant pour pouvoir demander de l'aide à votre tour.

## Fonctionnalités Principales (Prototype)
*   **Ranking de Réactivité** : Algorithme priorisant les mentors rapides.
*   **Capture de Contexte** : Envoi rapide du contexte (texte/audio) avant la connexion.
*   **Session Timer** : Compte à rebours visuel strict.

## Tech Stack (Phase 1)
*   **Frontend** : Vite + React + TailwindCSS
*   **Backend** : Node.js + Express
*   **Base de données** : (À définir, PostgreSQL prévu)
